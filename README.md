# c-setup-tool-for-visual-code

## prerequisites

- gcc can be done from <https://www.w3schools.in/c-tutorial/install/>
- how to launch check out <https://code.visualstudio.com/docs/cpp/config-mingw>

## run and build

- dev

    ```bash
    ./scripts/run.sh dev
    ```

- build

    ```bash
    ./scripts/run.sh build
    ```
